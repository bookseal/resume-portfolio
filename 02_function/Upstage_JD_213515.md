# AI Solution Engineer - Cloud

**URL:** https://careers.upstage.ai/ko/o/213515

## 직무 상세

업스테이지는 AI ​기술로 ​해결할 ​수 있는 ​비즈니스 문제들을 풀어갑니다. "Making ​AI ​Beneficial"이라는 미션 ​아래, 누구나 AI ​기술의 혜택을 ​누릴 ​수 있도록 ​'적용이 ​쉬운 ​AI 솔루션'을 개발하고 ​있습니다. ​세계 최고 수준의 ​AI ​모델을 ​다양한 분야에서 손쉽게 ​활용할 수 ​있는 ​API 시리즈를 ​출시하여 기업 ​고객들의 ​비즈니스 성공에 기여하고 ​있습니다. 또한, ​LLM 기술을 비즈니스 환경에 맞게 최적화하여 기업들의 업무 효율과 생산성을 높일 수 있는 Private LLM 서비스를 제공하는 등 AI가 세상에 이롭게 사용될 수 있도록 노력하고 있습니다.


LLM 기술과 더불어 Document Parse와 Information Extraction 등의 제품을 통해 문서 처리 분야에서 혁신을 이루고 있으며, 자체 LLM 및 Chat AI 기술을 비즈니스 환경에 맞게 최적화하여 기업들의 업무 효율과 생산성을 높일 수 있는 AI Space를 개발하는 등 업무 환경의 미래를 위한 인텔리전스를 만들어 갑니다.


Cloud Business Service 조직은 업스테이지의 AI 제품이 클라우드 환경에서 ‘계약 가능한 상태’에서 ‘실제 업무에 내재화된 상태’까지 연결되는 모든 기술적 여정을 책임집니다.


우리는 단순한 인프라 구축을 넘어 글로벌 클라우드 생태계 내에서 우리 제품이 고객 비즈니스에 실질적인 가치를 안착시키도록 실행력을 발휘합니다. 또한 Private Cloud 및 SaaS 환경에서의 기술 실행 허브이자 운영 책임 조직으로 기능하는 핵심 포지션을 찾고 있습니다.


CBS Solution Engineer는 AI 모델이 고객의 프로덕션 환경에서 안정적으로 서빙되기까지의 전체 인프라 라이프사이클 — 컨테이너 오케스트레이션, GPU 리소스 최적화, CI/CD 파이프라인, 모니터링, 오토스케일링 — 을 설계하고 자동화하는 것이 이 역할의 본질입니다.


우리가 직면하는 기술 문제는 단순 배포가 아닙니다. 고객마다 다른 VPC 토폴로지, 보안 정책, 클라우드 리전 제약 위에서 LLM 서빙 인프라를 프로덕션 레벨로 올려야 합니다. 한 번 만든 구현이 템플릿이 되고, 그 템플릿이 다음 딜의 배포 속도를 결정합니다.


또한 이 역할은 단순 실행자가 아닙니다. Solution Architect 가 설계한 아키텍처의 기술적 실현 가능성을 검증하고, 현장에서 반복되는 배포 패턴을 제품 기능으로 내재화하도록 제품팀에 직접 제안합니다. 개인의 엔지니어링 역량이 팀 전체의 딜리버리 스케일을 결정하는 포지션입니다.


주요 업무
- PoC 환경 구축 및 딜리버리 자동화
고객별 PoC 환경 세팅, API 연동, 컨테이너 배포까지 — SA가 수동으로 반복하는 전체 과정을 IaC 템플릿과 에이전트 기반 자동화 도구로 만들어 없앱니다. 인프라 프로비저닝, Docker/K8s 배포 파이프라인, CI/CD 구성을 직접 설계하고 구현합니다. 대규모 엔터프라이즈 고객을 위한 성능 확장(스케일링) 뿐만아니라, 특정 고객 요구에 대응하는 맞춤형 서비스 제공도 이루어질 수 있습니다.


- 고객 환경 프로덕션 엔지니어링
업스테이지 제품을 고객의 VPC, 보안 정책, 리전 제약 위에서 프로덕션 레벨로 동작시킵니다. API 연동, GPU 리소스 최적화, 오토스케일링, 모니터링 구성을 직접 수행하고, 연동 과정의 기술 이슈를 1차 대응합니다. 반복되는 연동 패턴은 코드 템플릿과 가이드 문서로 자산화하여 팀 전체가 재사용합니다.


- 신규 제품 파이프라인 통합 및 커스텀 개발
새롭게 합류하는 제품·기술 스택을 분석하고, Upstage의 인프라·보안·서빙 기준에 맞게 리아키텍처링하여 기존 프로덕트 파이프라인에 통합합니다. 고객별 요건에 맞춘 커스텀 개발을 직접 수행하며, 단순 유지보수가 아닌 독립 상용 제품으로 만들어내는 전 과정을 책임집니다.


- CSP Marketplace 운영 & 파트너 Enablement
AWS·Azure Marketplace의 제품 리스팅 패키징, 버전 업데이트, 미터링 연동 상태를 관리하고, CSP 채널 온보딩 및 기술 셋업을 주도합니다. 파트너가 Runbook만 보고 독립 배포할 수 있도록 기술 패키지를 설계하고, 파트너 환경에서의 동작을 검증합니다.
