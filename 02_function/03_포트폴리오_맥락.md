# Function_JD · 03 포트폴리오 맥락

## 형식
프로젝트 카드형, 한글, PDF. 각 카드: 한 줄 요약 / 문제정의 / 해결 / DevRel 관점 / 스택 / 링크.

## 핵심 프레이밍
> **표지/도입에서 bit-habit.com을 "AI를 누구나 이해하게 만드는 교육 제품 허브"로 제시.** 모든 프로젝트가 한 클러스터(k3s, $0)에 살아있는 라이브 서비스라는 점도 강점.

## 구성 — 두 축으로 묶기

### ▣ Teach (가르친다) — 상단 배치 (이 직무의 핵심)
1. **Seoul APT — ML 10단계 로드맵** ★대표 (Q3 도입 훅과 연결)
   - 서울 아파트 실거래가로 휴리스틱(L1)→선형회귀→…→AutoML(L10). "면적→가격 일차함수"라는 가장 쉬운 직관에서 출발. seoul-apt.bit-habit.com
2. **Viz — 선형대수 시각화** ★대표
   - 어텐션/트랜스포머 이해용 행렬 직관, 벡터→고유값→SVD→임베딩, Manim+Streamlit 온디맨드. viz.bit-habit.com
3. **Cloud Architect's Playbook (Wiki)** — 완성·공개
   - 기술 가이드북: Foundational/Systems Math, AI Agent Engineering, Cloud/M365 Playbook. 지속적 콘텐츠 저술 증거. wiki.bit-habit.com
4. **Munhaepang / Snowball English** — 중학생 AI 리터러시 / AI 영어학습 제품 (교육 도메인 폭)
5. **Daily Seongsu** — MLOps 레벨 1~10 가이드북 (AutoML Gradio 실행)
   - *(+ 초등수학 유튜브 200여 편: "기초 원리부터 가르친" 출발점으로 1줄 언급, 노출 OK)*

### ▣ Build (만든다)
6. **BookToss (Solar 재구축)** ★Upstage 제품 연계 · **진행 중**
   - 서울 25개 구 도서관 무API 통합검색(검색 5분→1분). 기존 LangGraph+Browser-use → **Solar API로 처음부터 재구축(오늘~약 1주, 6/28 전)**.
   - **빌드 타임라인/히스토리 섹션 포함**: Day별 계획(문제 재정의→Solar 연동→에이전트 흐름→검증). 추후 실제 커밋 history로 채워질 자리. 마치 쿡북처럼.
7. **Sentinel** — 메가존클라우드 해커톤 1등, 실시간 음성 감지, 로컬 NumPy $0·<10ms
8. **Bithabit** — IITP 해커톤 2등, 2년+ 실사용, 타임랩스+실시간채팅
9. **Quali-fit (현직)** — 사내 HR·자격증 추천, C레벨 도메인 지식 시스템화. quali-fit.bit-habit.com
