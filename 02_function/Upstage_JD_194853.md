# [독파모] AI Research Engineer - Post-training Internship (체험형)

**URL:** https://careers.upstage.ai/ko/o/194853

## 직무 상세

업스테이지는 "Making ​AI ​Beneficial"이라는 ​비전과 "Artificial ​General Intelligence (AGI) for ​Work"라는 ​미션을 바탕으로, ​AI 기술을 통해 ​인류가 일하는 ​방식을 ​혁신하는 기업입니다. ​단순한 ​업무 ​자동화를 넘어 복잡한 ​의사결정을 ​지원하고 기업의 비용 ​구조를 ​개선함으로써, ​구성원이 보다 가치 ​있는 일에 ​집중할 ​수 있는 ​환경을 만들어가고 ​있습니다.


이러한 ​비전을 실현하기 위해 ​업스테이지는 AGI의 ​핵심 기반 기술인 Foundation 모델을 지속적으로 발전시키고 있습니다. Global Frontier에서 추적하는 벤치마크 지표를 통해 모델 성능을 객관적으로 진단하고 개선하는 한편, 고객사의 실제 업무 환경에서 도출한 Workspace Benchmark Set을 자체적으로 구축하여 모델의 실용성과 성능을 함께 검증하고 있습니다.


이를 통해 업스테이지는 산업 현장의 복잡한 문제를 해결함과 동시에, 글로벌 수준의 기술 표준을 선도해 나가고 있습니다. 모델의 성능이 비약적으로 도약하는 그 순간은 자주 오지 않습니다. 그 티핑 포인트를 함께 넘어설, 지금 가장 빠른 성장 곡선을 함께 그릴 인재를 기다립니다.


Post-training 팀은 아래의 5가지 Skill에서의 Super Intelligence를 목표로 합니다.
(1) General Agent - 일반적 Agent 활용 시나리오 (built-in search 포함)에서 신뢰할만한 Agentic Behavior를 갖추는 것을 목표로 합니다.  
(2) Science Agent -  기본적인 STEM QA를 넘어서 Scientist들의 업무 효율을 개선할 수 있는 Agent Capability를 갖추는 것을 목표로 합니다.
(3) Coding Agent - SWE patch update 시나리오를 넘어서 NL2Repo까지 포함한 Coding 능력에 집중합니다. Foundation 모델 개발 코드 자체를 스스로 컨트롤하고 개선할수 있을 정도의 수준을 갖추는 것을 목표로 합니다.
(4) Office Work Agent - 실제 Office 환경에서 발생할 수 있는 모든 상황에 대응할 수 있는 Agent를 목표로 합니다. 다양한 환경, 다양한 input, 목표로 하는 output 등을 office 환경에 특화하여 고민하고 능력을 향상시킵니다. 단순 Level 1Task (1차원적 업무)를 넘어서 복합적인 상위 테스크인 Level 2테스크를 해결할 수 있는 성능을 갖추는 것을 목표로 합니다.
(5) Computer Use Agent - 기본적인 VLM 성능을 넘어, Computer screen과 interaction을 하며 다양한 테스크를 수행할 수 있는 Agent 성능 확보를 목표로 합니다.
이 핵심 목표를 달성하기 위하여, 확장 가능한 Agent 환경 구축 방법론, 초고품질 데이터 필터링 시스템, RLHF·RLVR·OPD 등 최신 학습 기법을 활용해 세계 최고 수준의 사후 학습 기술 개발을 주도합니다. 이 팀에 합류한다면, 진화하는 Foundation 모델 기술을 선도하며 산업 현장의 문제를 해결하는 차세대 기술 혁신을 함께 실현할 수 있을 것입니다.
