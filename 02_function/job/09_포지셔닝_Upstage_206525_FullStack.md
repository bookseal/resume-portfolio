# 포지셔닝 전략: Full-Stack Product Engineer Internship (Upstage 206525)

> 이 문서는 `Upstage_JD_206525.md` 직무에 맞춘 개인화 지원 전략입니다.

## 1. JD 핵심 요구사항 및 분석
이 직무는 연구용 AI 모델을 넘어, **사용자가 직관적으로 다룰 수 있는 대시보드 및 워크플로우 UI/UX를 개발할 풀스택 엔지니어**를 찾습니다.
- **핵심 키워드:** UI/UX 고도화, 대시보드(데이터 시각화), Next.js, Human-in-the-loop 피드백 루프

## 2. 포지셔닝 (한 줄 요약)
**"복잡한 AI(LLM, RAG) 기술을 Next.js와 Streamlit 같은 풀스택/시각화 도구를 활용해 사용자가 쓰기 편한 '진짜 제품'으로 깎아내는 엔지니어"**

## 3. 보유 자산(Asset) 매핑
| JD 요구사항 | Chan님의 핏(Fit) 무기 | 어필 포인트 |
| :--- | :--- | :--- |
| **대시보드 / 시각화** | **Quali-fit 사내 시스템**, **Seoul APT Prediction (Streamlit)** | 엑셀 수작업을 자동화하고, 복잡한 ML 데이터(행렬, 선형회귀)를 Streamlit으로 시각화하여 사용자가 직관적으로 보게 만든 경험 |
| **UI/UX & 워크플로우 빌드** | **BookToss (LangGraph)**, **Munhaepang (Next.js)** | 도서관 검색이라는 복잡한 워크플로우를 LangGraph 에이전트로 뒤단에서 자동화하고, 이를 Next.js 풀스택으로 매끄럽게 연결한 풀사이클 경험 |
| **초고효율 피드백 루프** | **가마솥 화상영어 4년**, **현직 FDE 경험** | 현업에서 경영진(C레벨)과 교사/학부모의 실제 피드백을 수집해 즉각적으로 서비스를 개선해 본 집요함 |

## 4. 이력서/자소서 작성 방향
- **경력 기술:** 단순 매니지먼트보다는 "직접 데이터를 다루고 사내 대시보드를 구축(Quali-fit)한 풀스택 개발 역량" 위주로 작성.
- **프로젝트 강조:** Next.js 풀스택(Munhaepang)과 Streamlit 시각화(Seoul APT)를 BookToss와 함께 최상단에 배치하여 프론트엔드/시각화 역량 증명.
- **오버스펙 이슈 방어:** "이미 기획부터 인프라까지 다 해본 생태계 파괴종 주니어"임을 어필하여 체험형 인턴이더라도 압도적인 성과를 낼 것임을 강조.
